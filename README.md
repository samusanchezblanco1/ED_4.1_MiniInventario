# MiniInventario

Aplicación de consola en Java para practicar Git y GitHub.

## Requisitos

* Java
* IntelliJ IDEA
* Git

## Ejecución

Abrir el proyecto en IntelliJ IDEA y ejecutar la clase Main.



Mejora del readme

Cambio Pull REQUEST

## Uso del programa
1. Ejecuta el programa desde IntelliJ.
2. Selecciona una opción del menú.
3. Puedes añadir productos y listarlos por pantalla.
4. El proyecto está pensado para practicar estructuras básicas y control de versiones.
